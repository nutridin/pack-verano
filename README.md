# Pack Verano

Landing page de venta del producto digital **Pack Verano** de Diletta Ghiglione (PDF de 38 páginas).
Página estática (HTML/CSS), lista para desplegar en Vercel, Netlify o cualquier hosting.

## Estructura

- `index.html` : landing page completa (lean, mobile-first, un único archivo).
- `gracias.html` : página de descarga tras el pago.
- `assets/` : imágenes (portada del pack y foto de la autora).
- `vercel.json` : configuración de despliegue.
- `LEEME.md` : guía para activar el pago (PayPal/Stripe) y la entrega del PDF.

## Puesta en marcha

Abre el bloque `CONFIG` al final de `index.html`, pega tu enlace de PayPal y ajusta el precio. Listo.
Más detalles en `LEEME.md`.
